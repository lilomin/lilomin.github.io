Docker学习

2017-04-17 • 标签: docker

简介

Docker 是一个可移植的容器，为开发者提供一种快速部署环境的解决方案，可以将相同的环境移植到任何linux机器上正常运行，而不需要重新搭建基础服务配置环境，省去了许多重复搭建工作，并且可以做成shell脚本一键部署，或者利用curl调用docker提供remote api实现远程部署，公司的线上环境使用到docker部署环境，因此自己也花了点时间去学习了一下；

搭建Docker

安装

首先你需要一台linux主机，听说docker(CE)已经支持windows10了，但是作为主机提供服务windows还是不靠谱，因此这里建议如果你的机器是windows的先去下个VMware再安装个linux；

配置Docker远程端口

添加一行开启tcp端口: OPTIONS=’–selinux-enabled -H tcp://0.0.0.0:2375 -H unix:///var/run/docker.sock’, 然后重启docker即可使用curl等调用docker提供的API, 不用登陆服务器或虚拟机就能通过脚本一键快速启停服务

启动Docker

安装配置完毕后，启动docker服务

Docker命令

学习docker命令前，我们需要先了解docker的基础理论，docker中有一个镜像的概念，顾名思义即是一个副本，可以将运行的进程打包成一个docker镜像，docker就是运行这些镜像的容器

docker pull

拉取docker镜像，可指定镜像仓库以及版本，默认docker.io里的latest

docker run

运行docker镜像

docker ps

查看docker运行镜像
查看所有启动过的镜像，包括挂掉的镜像

docker stop、docker restart、docker rm

docker镜像的启停与进程的移除
移除所有停掉的镜像进程

总结

总体来说docker对于开发者来说是非常方便，快速的环境搭建与启停，能为开发人员节省不少开发时间的，最后，这篇分享docker只是一个入门，满足基本日常使用，还有更多丰富的功能等待以后深入学习~
