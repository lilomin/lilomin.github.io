2016-08-26 • 标签: hexo

Hexo

hexo是一款基于node.js的静态博客框架，瞬间生成上百个静态文件，并且在github上有非常多的样式可供选择，有能力的完全可以自己定义自己样式，或者扩展已有的样式，hexo加上github pages实现一键部署。

之前一直想着自己写个博客，自己同时开发前后端，一步步的调样式以及配置后台服务器，实在是太麻烦，后来在网上看到hexo就开始在自己电脑上搭了一下环境，简单的几步就完成了一个简单的博客网站。

坑以及解决

配置好了后hexo deploy没反应

可能1: deploy 下面的各种key后面必须接上一个空格
可能2: hexo生成的.deploy_git有问题，删掉之后重新hexo g后再次部署

每次deploy都要手动输入用户密码

需要在本地配置ssh key，详见引用资源里

hexo deploy报错

确定git config配置有配置正确的用户名和邮箱

配置github ssh key
本地生成ssh key，执行命令一直回车

将生成的公钥配置到github ssh key列表
