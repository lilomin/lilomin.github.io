idea快捷操作记录

2016-12-20 • 标签: java

持续更新, 喂丸待续..

键入psv 生成main函数
键入sout 自动生成System.out.println();
键入psfs public static final String

以下快捷键都是重设置过的, 请自行 keymap 查询 key 对应的快捷键进行设置
Paste from History : Ctrl+Shift+V
Find in Path : Ctrl+Shift+F/Ctrl+H
Find Usages : Ctrl+G
Recent Files : Ctrl+E
Generate : Ctrl+Shift+S
Reformat Code : Ctrl+Shift+L
Method Hierarchy(方法层级结构) Ctrl+Shift+H
生成代码块 Ctrl+Alt+T
查找类 Ctrl+Shift+T

另外还有非常实用的两个快捷键
Search Everywhere : 双击Shift
Enter action or opion name : Ctrl + Shift + A
第一个是全局搜索任意文件名
第二个执行idea定义的指令，如generate、import、debug等等
